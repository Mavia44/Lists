Ordered List

An ordered list is a type of list in which items or elements are arranged in a specific sequence or order, usually indicated by numbers, letters, or other sequential markers.
Each item in the list is assigned a distinct ordinal number or letter to signify its position in the sequence.

Unordered List

An unordered list is a type of HTML markup used to create a list of items in no particular order. In an unordered list, each item is typically represented by a list item tag.
The list items are usually displayed with a bullet point or some other marker, indicating that the items in the list are not prioritized or arranged in a specific sequence.
